# vue2.x-SPA-webapp

> 这是通过对自己的[gulp-multipage](https://github.com/li2568261/gulp-multipage)项目进行的SPA重构，对vue2.x进一步的熟悉过程。

做到了：

1.通过vue对页面的渲染
2.页面切换滑动效果
3.axios数据请求
4.单个会话下本地数据存储
5.async/await处理异步函数
6.promise构造异步函数

## Build Setup

``` bash
# 依赖安装
npm install

# 进入开发模式
npm run dev

# 静态页面构建
npm run build

```
